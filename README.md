# Inquiry

**Corporate Repository** for Inquiry Institute official documents, charter, and corporate records.

## ?? Purpose

This repository contains:
- **Corporate Charter** - Official Institute charter and bylaws
- **Board Documents** - Board of Directors meeting minutes, resolutions, and records
- **Corporate Records** - Official filings, registrations, and legal documents
- **Governance Documents** - Policies, procedures, and governance frameworks
- **Official Communications** - Public statements and official communications
